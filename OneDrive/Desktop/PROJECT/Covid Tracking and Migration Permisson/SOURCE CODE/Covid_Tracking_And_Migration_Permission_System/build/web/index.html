INTRODUCTION

The Covid tracking and migration permission system provides its users to get online information about Total number of corona cases and number of deaths and recovered from corona virus. This system also allows the user to apply migration requisition form via internet. The main feature of the system is, it includes information retrieval facility to all the user. The user can get enough information about the number of corona cases and safety precaution to be taken and status of their migration requisition. In addition, user can also check their corona test results.The system allows the retrieving facility but also the updating facilitiesto the authorized persons.
